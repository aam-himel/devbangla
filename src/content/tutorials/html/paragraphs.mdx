# HTML Paragraphs

The HTML  `<p>`  tag is used to create paragraphs. For example,

```
<p>HTML is fun to learn.</p>
```

সাধারনত ওয়েব পেজে কোন অনুচ্ছেদ বা Paragraph প্রদর্শন করার জন্য প্যারাগ্রাফ ট্যাগ ব্যবহার করা হয়। অর্থাৎ এইচটিএমএল ডকুমেন্টে প্যারাগ্রাফ লেখা হয় ট্যাগ ব্যবহার করে।

`<p>` ট্যাগ দিয়ে শুরু করার পর প্যারাগ্রাফ এর শেষে ক্লোজিং ট্যাগ অর্থাৎ `</p>` দিতে হবে। যদিও অনেক ব্রাউযার এইচটিএমএল ডকুমেন্ট সঠিক ভাবেই প্রদর্শন করে, আপনি ক্লোজিং ট্যাগ অর্থাৎ দিতে ভুলে গেলেও। তারপরও আপনার উচিৎ সবসময় ক্লোজিং ট্যাগ ব্যবহার করা।

### উদাহরণ

```html
<body>
<p> This is a paragraph.</p>
</body>
```

## এইচটিএমএলএ নন-ব্রেকিং স্পেস ( &nbsp; ) এর ব্যবহার

এইচটিএমএল এ একাধিক স্পেস, কারেজ রিটার্ন, ট্যাব, ইত্যাদি পরিবর্তিত হয়ে শুধুমাত্র একটি স্পেস সৃষ্টি করে। আপনি code এ যতগুলোই স্পেস দিন না কেন আউটপুট এ মাত্র একটি স্পেসই পাবেন।

### উদাহরণ

```html
<body>
<p> This     is a     paragraph. </p>  
</body>
```

তাই আপনি যদি কোন text এর মাঝে অতিরিক্ত স্পেস তৈরি করতে চান তবে আপনাকে  **&nbsp;** শব্দ টি ব্যবহার করতে হবে। আপনি যতগুলো স্পেস চান আপনাকে ততগুলো &nbsp ব্যবহার করতে হবে।

### উদাহরণ


```html
<body>   
<p> This &nbsp; &nbsp; is a &nbsp; &nbsp; paragraph. </p>   
	  
</body>
```
